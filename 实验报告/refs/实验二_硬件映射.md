# 实验二 硬件映射（STM32F103C8T6 + OLED + W25Q64）

## 引脚分配

| 功能 | 引脚 | 代码宏 | 说明 |
|------|------|--------|------|
| KEY0 | PB11 | `KEY0` = `PBin(11)` | 调温度上限，内部上拉，低电平有效 |
| KEY1 | PB1 | `KEY1` = `PBin(1)` | 调光照上限，内部上拉，低电平有效 |
| OLED I2C SCL | PB6 | `OLED_W_SCL()` | 软件 I2C 时钟 |
| OLED I2C SDA | PB7 | `OLED_W_SDA()` | 软件 I2C 数据 |
| W25Q64 SS | PA4 | `MySPI_W_SS()` | 软件 SPI 片选，低电平有效 |
| W25Q64 SCK | PA2 | `MySPI_W_SCK()` | 软件 SPI 时钟 |
| W25Q64 MISO | PA3 | `MySPI_R_MISO()` | 软件 SPI 主入从出 |
| W25Q64 MOSI | PA1 | `MySPI_W_MOSI()` | 软件 SPI 主出从入 |
| USART1 TX | PA9 | — | 串口发送（可选，调试用） |
| USART1 RX | PA10 | — | 串口接收（可选，调试用） |
| DS0（LED0） | PB5 | — | 实验一用，实验二未使用 |
| DS1（LED1） | PB0 | — | 实验一用，实验二未使用 |

## DAP mini 调试器接线（同实验一）

| DAP mini | STM32 | 用途 |
|----------|-------|------|
| 3.3V | 面包板正极 | 给面包板供电 |
| GND | 面包板负极 | 共地 |
| SWDIO | PA13 | 数据 |
| SWCLK | PA14 | 时钟 |
| TX / RX | 不接 | 实验二不需要串口 |

## OLED 显示屏接线（I2C，4 脚模块）

模块引脚顺序：**GND, VCC, SCL, SDA**

| 模块引脚 | 接法 |
|---------|------|
| GND | → 面包板 GND |
| VCC | → 面包板 3.3V |
| SCL | → PB6 |
| SDA | → PB7 |

> ⚠️ VCC 和 GND 接面包板电源轨，**不是**接 PB4/PB5 GPIO。I2C 地址 0x78。

## W25Q64 存储模块接线（软件 SPI，6 脚模块）

模块引脚顺序：**VCC, CS, DO, GND, CLK, DI**

| 模块引脚 | 接法 |
|---------|------|
| VCC | → 面包板 3.3V |
| CS | → PA4 |
| DO（MISO） | → PA3 |
| GND | → 面包板 GND |
| CLK（SCK） | → PA2 |
| DI（MOSI） | → PA1 |

> ⚠️ VCC 和 GND 接面包板电源轨。DO 和 DI 容易接反——DO 对应 MISO，DI 对应 MOSI。

## 按键接法

```
引脚 → 按键一脚，另一脚 → GND
```

- KEY0: PB11 → 按键 → GND
- KEY1: PB1 → 按键 → GND

> 代码中已配置内部上拉，按键不按时引脚为高电平，按下接地为低电平。
