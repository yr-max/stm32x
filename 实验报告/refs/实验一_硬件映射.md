# 实验一 硬件映射（STM32F103C8T6 最小系统板）

## 引脚分配

| 功能 | 引脚 | 代码宏 | 说明 |
|------|------|--------|------|
| DS0（LED0） | PB5 | `LED0` = `PBout(5)` | 低电平点亮 |
| DS1（LED1） | PB0 | `LED1` = `PBout(0)` | 低电平点亮 |
| KEY0 | PB1 | `KEY0` = `PBin(1)` | 内部上拉，低电平有效 |
| KEY1 | PA1 | `KEY1` = `PAin(1)` | 内部上拉，低电平有效（原 PB2 未引出） |
| USART1 TX | PA9 | — | 串口发送，接 USB-TTL RXD |
| USART1 RX | PA10 | — | 串口接收，接 USB-TTL TXD |

## DAP mini 调试器接线

| DAP mini | STM32 | 用途 |
|----------|-------|------|
| 3.3V | 3.3V | 供电 |
| GND | GND | 共地 |
| SWDIO | PA13 | 数据 |
| SWCLK | PA14 | 时钟 |
| TX | PA10 | 串口发送（可选，DAP mini 内置串口时） |
| RX | PA9 | 串口接收（可选） |

## LED 接法

```
引脚 → 限流电阻(220Ω~1kΩ) → LED 长脚(+) → LED 短脚(-) → GND
```

- DS0: PB5 → 电阻 → 长脚 → 短脚 → GND
- DS1: PB0 → 电阻 → 长脚 → 短脚 → GND

## 按键接法

```
引脚 → 按键一脚，另一脚 → GND
```

- KEY0: PB1 → 按键 → GND
- KEY1: PA1 → 按键 → GND

> 代码中已配置内部上拉，按键不按时引脚为高电平，按下接地为低电平。
