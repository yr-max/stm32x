# 实验五 硬件映射

## 1. 主控

STM32F103C8T6（Medium Density），72MHz，编译宏 `STM32F10X_MD`，启动文件 `startup_stm32f10x_md.s`。

## 2. 引脚分配

与实验四保持一致，无新增引脚。

| 功能 | 引脚 | 复用/说明 |
|------|------|-----------|
| LED0 / DS0 | PB5 | 低电平点亮 |
| LED1 / DS1 | PB0 | 低电平点亮 |
| KEY0 | PB11 | 上拉输入 |
| KEY1 | PB1 | 上拉输入 |
| 串口 TX | PA9 | USART1，115200 |
| 串口 RX | PA10 | USART1 |
| 热敏 ADC | PA0 | ADC1_CH0 |
| 光敏 ADC | PA5 | ADC1_CH5 |
| OLED SCL | PB6 | I2C 模拟 |
| OLED SDA | PB7 | I2C 模拟 |
| W25Q64 CS | PA4 | 软件 SPI |
| W25Q64 DO | PA3 | 软件 SPI |
| W25Q64 DI | PA2 | 软件 SPI |
| W25Q64 CLK | PA1 | 软件 SPI |
| 舵机 PWM | PA6 | TIM3_CH1，50Hz |
| 电机 PWM | PB10 | TIM2_CH3（PartialRemap2），10kHz |
| 电机 IN1 | PB8 | TB6612FNG 方向 |
| 电机 IN2 | PB9 | TB6612FNG 方向 |

## 3. 注意事项

- PA0 同时作为热敏 ADC 和 WK_UP（KEY_UP）引脚；本实验未使用 WK_UP，仅做 ADC 采样。
- PA1~PA4 已被 W25Q64 软件 SPI 占用，舵机/电机 PWM 使用 PA6/PB10 以避开冲突。
- TIM2_CH3 默认在 PA2，通过 `GPIO_PartialRemap2_TIM2` 映射到 PB10。
