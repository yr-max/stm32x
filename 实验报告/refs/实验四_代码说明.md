## 控制逻辑

### 光照 → 舵机角度

```c
g_servo_angle = (uint8_t)((uint32_t)g_light * 180 / 100);
Servo_SetAngle(g_servo_angle);
```

- 光照 0~100% 线性映射到舵机角度 0~180°
- 内部公式：`Compare = 500 + Angle * 2000 / 180`

### 温度 → 电机转速

```c
if (g_temp > g_temp_limit)
{
    int16_t over = g_temp - g_temp_limit;
    g_motor_speed = over * 10;          /* 每超 1°C +10 转速 */
    if (g_motor_speed > 100) g_motor_speed = 100;
}
else
{
    g_motor_speed = 0;
}
Motor_SetSpeed(g_motor_speed);
```

- 温度超过上限后启动电机
- 每超 1°C 转速加 10，最高 100
- `Motor_SetSpeed(0)` 时 AIN1=AIN2=0，电机停止

## 关键修复记录

| 问题 | 根因 | 修复 |
|------|------|------|
| 电机不转，M=100 但 PB10 无输出 | TIM2_CH3 remap 值错误，用了 `PartialRemap1` | 改为 `GPIO_PartialRemap2_TIM2` |
| 驱动模块型号误判 | 看到大芯片和 16 个引脚以为是 L298N | 实际为 TB6612FNG，引脚名称完全不同 |
| 电机铜片接法 | 电机没有杜邦线接口 | 用鳄鱼夹夹住两个铜片 |

## 注意事项

1. **TIM2_CH3 到 PB10 必须用 `GPIO_PartialRemap2_TIM2`**，PartialRemap1 只会移动 CH1/CH2。
2. **TB6612FNG 的 STBY 必须接 3.3V**，否则整个 A/B 通道都不工作。
3. **VM 必须接 5V** 给电机供电，VCC 只给芯片逻辑供电。
4. 电机铜片正反接只影响转向，可通过交换 AO1/AO2 或在代码中反转方向修正。
