# 实验三 硬件映射（STM32F103C8T6 + 热敏 + 光敏）

## ADC 引脚分配

| 传感器 | 模块引脚 | STM32 引脚 | ADC 通道 | 说明 |
|--------|----------|------------|----------|------|
| 热敏电阻 | AO | PA0 | ADC_Channel_0 | NTC 10K B=3950 |
| 光敏电阻 | AO | PA5 | ADC_Channel_5 | 线性映射 0~100% |

> 选 PA0/PA5 因为 PA1~PA4 已被 W25Q64 软件 SPI 占用。

## 传感器模块接线（MH-Sensor-Series，4 脚）

模块引脚顺序：**VCC, GND, AO, DO**

| 模块引脚 | 接法 | 说明 |
|---------|------|------|
| VCC | → 面包板 3.3V | |
| GND | → 面包板 GND | |
| AO | → PA0（热敏）/ PA5（光敏） | 模拟输出，接 ADC |
| DO | 悬空不接 | 数字阈值输出，实验三未使用 |

## 温度转换公式

**NTC 参数**：R25=10KΩ, B=3950K, 分压电阻=10KΩ（上臂）

```
电路：VCC(3.3V) → 10K → AO(ADC) → NTC → GND

Vout   = ADC × 3.3 / 4095
R_ntc  = 10000 × Vout / (3.3 - Vout)
T(K)   = 1 / (1/298.15 + ln(R_ntc/10000) / 3950)
T(°C)  = T(K) - 273.15
```

> NTC 负温度系数，温度升高 → 阻值降低 → AO 电压降低 → ADC 值降低。

## 光照转换

```
光照% = (4095 - ADC_RAW) × 100 / 4095
```

> 传感器在分压下臂，翻转后映射到 0~100%。
