# 实验四 硬件映射（STM32F103C8T6 + 舵机 + 直流电机）

## 新增引脚分配

| 功能 | 引脚 | 定时器/类型 | 代码宏/函数 | 说明 |
|------|------|------------|------------|------|
| 舵机 PWM | PA6 | TIM3_CH1 | `Servo_SetAngle()` | 50Hz，脉冲 500~2500μs |
| 电机 PWM | PB10 | TIM2_CH3 | `Motor_SetSpeed()` | 10kHz，需 `PartialRemap2` |
| 电机方向 IN1 | PB8 | GPIO 输出 | `GPIO_SetBits/ResetBits` | TB6612FNG AIN1 |
| 电机方向 IN2 | PB9 | GPIO 输出 | `GPIO_SetBits/ResetBits` | TB6612FNG AIN2 |

> 舵机 PWM 在 PA6（TIM3_CH1）无需 remap；电机 PWM 在 PB10 必须配置 `GPIO_PartialRemap2_TIM2`。

## 继承接线（同实验三，不变）

| 功能 | 引脚 |
|------|------|
| 热敏电阻 AO | PA0 |
| 光敏电阻 AO | PA5 |
| W25Q64 SS/SCK/MISO/MOSI | PA4/PA2/PA3/PA1 |
| OLED I2C SCL/SDA | PB6/PB7 |
| KEY0/KEY1 | PB11/PB1 |
| LED0/LED1 | PB5/PB0 |
| USART1 TX/RX | PA9/PA10 |

## 舵机 SG90 接线

模块引脚顺序：**信号(橙), VCC(红), GND(棕)**（颜色可能因厂商略有差异，以实物为准）

| 舵机引脚 | 接法 |
|---------|------|
| 信号（通常为橙色） | → PA6 |
| VCC（通常为红色） | → 面包板 3.3V |
| GND（通常为棕色） | → 面包板 GND |

> 舵机臂作为可视指针，光照 0~100% 映射到 0~180°。

## TB6612FNG 电机驱动模块接线

模块左排引脚：**PWMA, AIN2, AIN1, STBY, BIN1, BIN2, PWMB, GND**
模块右排引脚：**VM, VCC, GND, AO1, AO2, BO1, BO2, GND**

| TB6612FNG 引脚 | 接法 | 备注 |
|---------------|------|------|
| AIN1 | → PB8 | 方向控制 |
| AIN2 | → PB9 | 方向控制 |
| PWMA | → PB10 | PWM 调速 |
| VCC | → 面包板 3.3V | 芯片逻辑供电 |
| STBY | → 面包板 3.3V | **使能脚，必须接高电平** |
| VM | → 面包板 5V | 电机动力供电 |
| GND | → 面包板 GND | 与 C8T6 共地 |
| AO1 | → 电机铜片1 | 输出 A 通道 |
| AO2 | → 电机铜片2 | 输出 A 通道 |

> BIN1, BIN2, PWMB, BO1, BO2 在本实验中悬空不接。

## 电机铜片接线

直流电机有两个裸露焊片（铜片），用鳄鱼夹或杜邦线固定：

```
TB6612FNG AO1 ──→ 鳄鱼夹 ──→ 电机铜片①
TB6612FNG AO2 ──→ 鳄鱼夹 ──→ 电机铜片②
```

> 两个铜片正反接只影响转向，不会损坏电机。
